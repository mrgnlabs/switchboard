## Learn

Before you begin building your first Switchboard Function, checkout the
following documents to learn more about the Switchboard network, technical
architecture and supported products.

**Technical Documentation**

- [The Oracle Problem](/the-oracle-problem): Why we need oracles.
- [Program Architecture](/architecture): About the network's structure and core
  components.
- [The Data Models](/data): About the different methods of data propagation
  (largely known as push, pull and callback).
- [What are TEEs?](/tee): About Trusted Execution Environments (TEEs) providing
  verifiable compute to your functions.

**Products**

- [Switchboard Functions](/functions): All about Switchboard Functions.
- [Switchboard Secrets](/secrets): Utilize web2 confidential data within your
  functions.
